# Skills

Installable AI agent skills.

## server-first-dev

One-time bootstrap skill for building and deploying a production app on a fresh server.

Install:

```bash
npx skills add https://github.com/Majncz/skills --skill server-first-dev
```

For the article flow, attach [`server-first-dev-bootstrap.md`](server-first-dev-bootstrap.md) to the first Cursor chat on the fresh SSH server and paste your application requirements into it.

Local development:

```bash
npx skills add /Users/jakubrana/skills-repo --skill server-first-dev
```
